- v0.1 Rendering
- v0.2 Search
- v0.3 Filter
- v0.4 Extension

## 1. Project Structure

  style.css
  app.js

## 2. Stage Rules
- Do NOT skip stages
- Do NOT implement future features early
- One small task per step

## 3. Context Rules
- Only use provided files
- Do NOT modify unrelated files

## 4. Code Rules
- Minimal changes only
- Keep existing structure
- No full rewrites

## 5. Features

### Search
- Real-time input
- Match name and city

### Filter
- Multi-select tags
- Combine with search

### Reset
- Clear all filters

## 6. Constraints

- No full file rewrite

## 7. Output Format
Use patch or clear change description

## Final Rule
You are a careful engineer, not a generator.
